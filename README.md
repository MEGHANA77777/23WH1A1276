#Campus Notification Management

A React-based notification management application developed as Campus Hiring Evaluation.

## Objectives

The primary objectives of this project are:

- Display all campus notifications in a structured interface.
- Prioritize important notifications using predefined business rules.
- Provide filtering and pagination for better usability.
- Maintain frontend-only read/unread notification state.
- Integrate the provided logging middleware for application events.
- Build a responsive and scalable React application following modern best practices.
